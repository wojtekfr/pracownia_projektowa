# pracownia
Po uruchomieniu programu bez parametrów, należy wprowadzić ścieżkę katalogu, w którym znajdują się pliki Excela (z rozszerzeniem xls lub xlsx).
Można także wywołać program z lini komend dodając parametr -filename i ścieżkę w cudzysłowiu.
Przykład
java -jar pracownia.jar -filename "c:\pracownia"
